### 🔹 1. 포인터란?

* **정의**: 포인터는 **다른 변수의 메모리 주소를 저장하는 변수**입니다.
* 예시:

  ```c
  int a = 10;
  int *p = &a;  // p는 a의 주소를 저장하는 포인터
  ```

---

### 🔹 2. 기본 용어

| 용어       | 의미                          |
| -------- | --------------------------- |
| `&`      | 주소 연산자 (변수의 주소를 얻음)         |
| `*`      | 간접 참조 연산자 (포인터가 가리키는 값을 얻음) |
| `int *p` | int형 변수를 가리키는 포인터           |

---

### 🔹 3. 간단한 예제

```c
#include <stdio.h>

int main() {
    int a = 5;
    int *p = &a;

    printf("a의 값: %d\n", a);           // 5
    printf("a의 주소: %p\n", &a);        // 주소값
    printf("p의 값(=a의 주소): %p\n", p);
    printf("*p의 값(=a의 값): %d\n", *p); // 5

    return 0;
}
```
