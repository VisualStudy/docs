# C 언어 배열과 구조체의 복사 원리

---

## 1. 개요

C 언어에서 **배열과 구조체는 메모리상에 연속적으로 저장**되지만,  
**복사 방식에는 근본적인 차이**가 있다.

- 배열(Array): **대입 연산 불가**
- 구조체(Struct): **같은 타입끼리 대입 연산 가능**

이 문서는 그 이유와 원리를 코드와 함께 정리한다.

---

## 2. 배열(Array)의 복사

### 2.1 배열은 대입할 수 없다

```c
char src[5] = "abcd";
char dst[5];

dst = src;   // ❌ 컴파일 오류
````

#### 이유

* 배열 이름은 **주소 상수**
* 주소 자체를 변경하는 대입 연산(`=`)이 허용되지 않음
* 배열은 **값 타입이 아님**

---

### 2.2 배열은 원소 단위로 복사해야 한다

#### 반복문을 이용한 복사

```c
int i = 0;
while (src[i] != '\0') {
    dst[i] = src[i];
    i++;
}
dst[i] = '\0';
```

#### `strcpy` 사용

```c
strcpy(dst, src);
```

> 문자열 복사는 `'\0'`를 만날 때까지 문자를 하나씩 복사하는 방식이다.

---

### 2.3 배열 복사의 핵심 정리

* 배열은 **대입 불가**
* 반드시 **원소 단위 복사** 필요
* `strcpy`, `memcpy` 등은 내부적으로 반복 복사 수행

---

## 3. 구조체(Struct)의 복사

### 3.1 구조체는 대입이 가능하다

```c
struct Point {
    int x;
    int y;
};

struct Point p1 = {10, 20};
struct Point p2;

p2 = p1;   // ✅ 구조체 전체 복사
```

#### 이유

* 구조체는 **하나의 값 타입**
* 컴파일러가 멤버 전체를 자동으로 복사

---

### 3.2 구조체 복사의 내부 동작 개념

```c
p2 = p1;
```

은 개념적으로 다음과 같다:

```c
memcpy(&p2, &p1, sizeof(struct Point));
```

---

### 3.3 구조체 안에 배열이 있는 경우

```c
struct Person {
    char name[10];
    int age;
};

struct Person a = {"Alice", 30};
struct Person b;

b = a;   // ✅ name 배열까지 함께 복사됨
```

* 구조체 내부의 배열도 **자동으로 함께 복사**
* 배열 단독일 때와 다른 동작

---

## 4. 구조체 복사의 주의점 (얕은 복사)

### 4.1 포인터 멤버를 가진 구조체

```c
struct Data {
    int *ptr;
};

int x = 10;
struct Data d1, d2;

d1.ptr = &x;
d2 = d1;
```

#### 결과

* `ptr`의 **주소 값만 복사**
* 두 구조체가 **같은 메모리**를 가리킴

```c
*d2.ptr = 20;  // x 값도 변경됨
```

> 구조체 대입은 기본적으로 **얕은 복사(shallow copy)** 이다.

---

## 5. 배열 vs 구조체 복사 비교

| 구분         | 배열    | 구조체    |
| ---------- | ----- | ------ |
| 대입 연산(`=`) | ❌ 불가  | ✅ 가능   |
| 복사 방식      | 원소 단위 | 통째 복사  |
| 값 타입 여부    | ❌     | ✅      |
| 내부 배열      | —     | 함께 복사  |
| 포인터 멤버     | —     | 주소만 복사 |

---

## 6. 시험 대비 핵심 문장

* 배열은 주소 상수이므로 대입 연산이 불가능하다.
* 구조체는 하나의 값 타입이므로 같은 구조체 타입끼리는 대입이 가능하다.
* 구조체 대입은 멤버를 바이트 단위로 복사하는 동작이다.
* 포인터 멤버를 포함한 구조체 복사는 얕은 복사이다.

---

## 7. 한 줄 요약

> 배열은 직접 복사할 수 없고,
> 구조체는 같은 타입끼리 대입을 통해 통째로 복사할 수 있다.
